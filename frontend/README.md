## setup

### prerequisites

- [node](https://nodejs.org/en)
- [pnpm](https://pnpm.io/installation)

### installation

1. clone the repo
2. `cp .env.example .env`, fill in the required environment variables
3. run `pnpm install`
4. run `pnpm run dev`
